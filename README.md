# Enterprise Network Security Lab

## 📌 Overview
This project simulates a real-world enterprise network using Cisco Packet Tracer, including firewall protection and an attacker scenario.

## 🌐 Network Topology
- Branch 1 (New York): PCs + Switch + Router
- Firewall (ASA)
- Internet Simulation (Router)
- Branch 2 (Tokyo): Router + Firewall + Switch + Servers
- External Attacker Device

## 🔐 Security Concept
- Network segmentation between internal and external networks
- Firewall used to control traffic
- Attacker simulation to test network security

## 🎯 Objectives
- Understand enterprise network structure
- Learn basic firewall and security concepts
- Practice real-world network simulation

## 🛠 Tools Used
- Cisco Packet Tracer

## 🚀 Future Improvements
- Implement firewall rules (ACLs)
- Add VLAN segmentation
- Simulate real services (HTTP, DNS)
